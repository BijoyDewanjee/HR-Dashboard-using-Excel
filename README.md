# 📊 HR Analytics Dashboard (Excel)

## 🔍 Overview

This project presents an **interactive HR Analytics Dashboard** built using Microsoft Excel.
The dashboard provides insights into workforce composition, salary analysis, recruitment trends, and employee demographics.

It is designed to help HR teams make **data-driven decisions** through clear visualizations and KPIs.

---

## 🎯 Key Features

### 📌 1. Workforce Composition

* Total Headcount
* Department-wise Employee Distribution
* Designation Breakdown
* Employee Category Ratio (Management vs Direct vs Indirect)

---

### 💰 2. Salary & Financial Analysis

* Total Monthly Payroll
* Average Salary by Department
* Salary Comparison by Designation
* Salary Difference (Permanent vs Probation)

---

### 📈 3. Recruitment & Retention

* Monthly Hiring Trends (Line Chart)
* Employee Status Distribution (Active vs Resigned vs Fire)
* Confirmation Pipeline (Probation vs Permanent)

---

### 👥 4. Employee Demographics & Logistics

* Gender Distribution by Department
* Commute Distance Analysis (1–3 km vs 10–15 km)
* Marital Status Distribution
* Blood Group Distribution (for emergency planning)

---

## 📊 Dashboard Components

### 🔷 KPI Cards

* Total Employees
* Total Basic Salary
* Active Employees
* Average Salary

---

### 📊 Visualizations

* Bar Chart → Employees by Department
* Pie Chart → Gender Distribution
* Donut Chart → Employee Type (Permanent vs Probation)
* Bar Chart → Average Salary by Designation
* Line Chart → Monthly Hiring Trends

---

### 🎛️ Interactive Features

* Slicers for dynamic filtering:
  * Department
  * Working Status
  * Gender
  * Employee Category

---

## 🛠️ Tools & Technologies

* Microsoft Excel
* Pivot Tables
* Pivot Charts
* Slicers
* Excel Formulas (COUNTIF, AVERAGEIF, SUM, etc.)

---

## 📁 Dataset

* HR Dataset (Employee-level data including salary, department, designation, etc.)

---

## 🚀 How to Use

1. Open the Excel file
2. Navigate to the **Dashboard sheet**
3. Use slicers to filter data dynamically
4. Analyze KPIs and charts for insights

---

## 🎯 Key Insights

* Workforce is primarily concentrated in operational roles
* Majority of employees live within short commuting distance
* Salary varies significantly across designations and employee types
* Hiring trends fluctuate across months

---

## 📌 Conclusion

This dashboard demonstrates how Excel can be used as a powerful tool for HR analytics, enabling better workforce planning and decision-making.
